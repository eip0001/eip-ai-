1
Open Remix IDE browser
Go to remix.ethereum.org — no install needed. Create a new workspace or use the default one.
2
Add EIPTestToken.sol
In the file explorer, create EIPTestToken.sol and paste this code:
// SPDX-License-Identifier: MIT // Paste full contents of contracts/EIPTestToken.sol pragma solidity ^0.8.24; contract EIPTestToken { string public name = "EIP Test Token"; string public symbol = "EIPTEST"; uint8 public decimals = 18; uint256 public totalSupply; mapping(address => uint256) public balanceOf; mapping(address => mapping(address => uint256)) public allowance; uint256 public constant MINT_LIMIT = 1_000_000 * 1e18; event Transfer(address indexed from, address indexed to, uint256 value); event Approval(address indexed owner, address indexed spender, uint256 value); function mint(address to, uint256 amount) external { require(amount <= MINT_LIMIT, "Exceeds mint limit"); totalSupply += amount; balanceOf[to] += amount; emit Transfer(address(0), to, amount); } function transfer(address to, uint256 amount) external returns (bool) { return _transfer(msg.sender, to, amount); } function transferFrom(address from, address to, uint256 amount) external returns (bool) { uint256 a = allowance[from][msg.sender]; if (a != type(uint256).max) { require(a >= amount); allowance[from][msg.sender] = a - amount; } return _transfer(from, to, amount); } function approve(address spender, uint256 amount) external returns (bool) { allowance[msg.sender][spender] = amount; emit Approval(msg.sender, spender, amount); return true; } function _transfer(address from, address to, uint256 amount) internal returns (bool) { require(balanceOf[from] >= amount); balanceOf[from] -= amount; balanceOf[to] += amount; emit Transfer(from, to, amount); return true; } }
3
Add EIPEscrow.sol
Create a second file EIPEscrow.sol and paste the full contents from contracts/EIPEscrow.sol in the zip.
4
Compile both contracts Solidity 0.8.24
Click the Solidity Compiler tab (left sidebar). Set compiler to 0.8.24. Click Compile EIPTestToken.sol, then Compile EIPEscrow.sol.
5
Connect MetaMask → Base Sepolia
In Remix's Deploy & Run tab, set Environment to "Injected Provider — MetaMask". Make sure MetaMask is on Base Sepolia (chainId 84532). Your address 0xdcD1…BDAe should appear.
6
Deploy EIPTestToken no constructor args
Select EIPTestToken in the contract dropdown. Click Deploy. Confirm in MetaMask. Copy the deployed address from the console — you'll need it below.
7
Deploy EIPEscrow 2 constructor args
Select EIPEscrow. Fill in the constructor arguments:
_usdcToken: 0x036CbD53842c5426634e7929541eC2318f3dCF7e // USDC on Base Sepolia _feeRecipient: 0xdcD11482abbD1F17F11e26A1193AfD4EED3CBDAe // your wallet (fees go here)
Click Deploy. Confirm in MetaMask. Copy the escrow contract address.
8
Mint test tokens to yourself
In Remix, expand the deployed EIPTestToken contract. Call mint with:
to: 0xdcD11482abbD1F17F11e26A1193AfD4EED3CBDAe amount: 1000000000000000000000000 // 1,000,000 EIPTEST (18 decimals)
9
Paste addresses below → save to market
Enter the deployed addresses and click Save & go live. This updates market.html in your browser — download the updated file.
